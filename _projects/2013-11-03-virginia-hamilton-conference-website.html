Kent State hosts the Virginia Hamilton Conference each year.  The conference organizers needed to replace an outdated Wordpress-based site with a new website in Kent State's content management system.

My role in this project:

  • Designed low fidelity and high fidelity wireframes
  • Designed a pattern library demonstrating fonts, page elements and colors for the client
  • Developed HTML and CSS in Kent State's content management system.  Developed the CSS using SASS/COMPASS using version control.
  • Worked with client to support content entry into the system
  • Successfully launched the website prior to the conference
